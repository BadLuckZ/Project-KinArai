# Kin-Arai (กินอะไร)

A web app that helps you decide what to eat by randomly picking a restaurant/dish, avoiding ones you've eaten recently. Keeps a history of past picks.

Built as a project to practice full-stack web development and real deployment on AWS.

## Sources

- [Bun](https://bun.sh)
- [Elysia](https://elysiajs.com)
- [Vue 3](https://vuejs.org)
- [Vite](https://vitejs.dev)

# Initialization

Frontend

```sh
bun create vite frontend --template vue
```

Backend

```sh
bun create elysia backend
```
